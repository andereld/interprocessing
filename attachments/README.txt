The naming of files in the processed_audio_files/ directory specify what
kind of processing has taken place. For a file named

  /dynamic-bp-ringmod-i-guitar-a-vocals-001_v1.wav

The processing which took place
  - was dynamic
  - used the bandpass-filter / ring modulator effect
  - the input signal was the guitar sample
  - the affecting signal was the vocal sample
  - the weighting was:
    * w_centroid = 0
    * w_cps = 0
    * w_rms = 1
  - it is the first (v1) of several files with these settings
